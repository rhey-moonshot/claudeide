ClaudeIDE

by Rhey Minoza

An editor built for vibe coding with Claude Code.

VS Code and the editors before it were made for a world where you type every line yourself. ClaudeIDE is made for the new one — where you conduct a swarm of Claude Code agents, each living in its own terminal, and watch them work shoulder to shoulder.

Open many sessions at once and tell at a glance which agents are working, which need you, and which are done — so you stay in flow instead of digging through stacked terminal tabs.

Free & open source — made with care, and shared with everyone. 🌱
